## 📖 Overview

Swiggy React Clone is a frontend food ordering application inspired by modern food delivery platforms. The project demonstrates component-based architecture, state management, cart functionality, and responsive UI design using React.

Users can explore food items, manage their cart, and place orders through an intuitive interface designed for a smooth user experience.

---

## ✨ Features

### 🍽️ Food Listings

* Browse a collection of food items
* View food details and pricing
* Clean and organized layout

### 🛒 Smart Cart System

* Add items to cart
* Remove items from cart
* Real-time cart updates
* Floating cart access bar

### 🔔 User Experience

* Interactive toast notifications
* Order confirmation messages
* Responsive design for all screen sizes
* Smooth and intuitive navigation

### ⚛️ React Architecture

* Functional Components
* React Hooks
* Context API for state management
* Reusable UI components

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/swiggy-react.git
```

### Navigate to Project Folder

```bash
cd swiggy-react
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

Application will run at:

```text
http://localhost:3000
```

---

## 📦 Production Build

Create an optimized production build:

```bash
npm run build
```

The build files will be generated inside the `build` folder.
